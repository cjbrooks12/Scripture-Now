This is the official webpage for Scripture Memory Notifications, an Android application to help Christians memorize Bible verses. This application is completely open-source. To obtain the source and start working on it for yourself, fork this repo and get to work!

Alternatively, if you don't want to help with this project you can simply download it from the Google Play Store

What is it?

I believe that Christians should strive to keep the Word of God in our heart and on our mind, and part of this is memorizing Scripture. This application puts an ongoing notification into your Android device's notification drawer, so that every time you pull it down, you see the verse you want to memorize and try to remember it. All verses are saved in a beautiful Google Now-inspired layout, allowing one to set specific verses as "current" or "memorized" and try to memorize those that aren't yet committed to memory. The app also includes a Verse of the Day feature, which displays a new Bible verse every day, courtesy of VerseOfTheDay.com.

How's it Doing?

Scripture Memory Notifications is currently in V2.4. Over the course of this summer, I have added many new features, include:

  • Verse of the Day notifications, and able to be shown in many different Bible translations
  • Homescreen and lockscreen widgets for both the main notification verse and the Verse of the Day
  • Change how the text in the notification is displayed (dashes instead of letters, only the first letter of each word, or a combination of both) for stronger memorization
  • Improved graphics and animations, to make the UX much more enjoyable
  • A dark theme in addition to the original light theme

Beyond that, the app has seen more than 2500 downloads and has an average rating of 4.2/5 from 33 reviews on the Google Play Store.

What am I working on now?

My main project for this lately has been a complete redesign of the data structures on the backend. Previously, the data was very simple, and while it did what I wanted it to do, that was really all it could do. I am creating a library called AndroidBibleTools which is seeks to change this by making the data more closely reflect Verses as we perceive them in the real world. This allows me to do much more with it, including:

  • Make getting and using Bible verses in any app dead-simple. A Verse object behaves exactly as you would expect: as a single verse from the Bible, which may be in any translation. Same with a passage, which is a collection of single verses
  • Create data structures that make using these verses in groups easy, and scalable. They will support searching, sorting, and dynamic loading of data to give what is needed quickly without overloading the memory buffer
  • Write Verses to a file in a way that is easily shared and recovered, not only with my app but with others. It is based on the format created by RememberMe, but builds on it to add just a bit more
  • Download verses from the Internet very easily, without needing to know anything about HTML retrieval and parsing
  • Share verses directly to other Android apps in a way that creates a unified connumity of Christian software. Any app can use this library to send a verse to any other app using the library, but lacking any apps to send it to using the library will send it as a normal verse to anything

This library has been my main focus for a while now, and it is as of yet unfinished, but a good portion of it is working correctly. As soon as I finish it, I will begin to contact other Android developers and companies to encourage them to join this new sharing system to bring us all together in one community.
