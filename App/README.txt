Scripture Memory Notifications 
	-For Android SDK 8+ (2.2 - present)
	-Source code can be found at https://github.com/cjbrooks12/scripturememory
	-Uses the ActionBarSherlock Library, which must be linked externally. ActionBarSherlock can be found at http://actionbarsherlock.com/
	-Created and maintained by:
		Casey Brooks
		cjbrooks12@gmail.com
	-This app was made to help memorize verses from the Bible by displaying a notification on your Android phone or tablet. App is developed entirely in Java, and data is stored using Android's SQLite database APIs. App uses a custom theme based on Android's Holo theme, but with the color #508a4c, and all elements should follow along with this theme.
